# Security

SharpMemory is currently designed for local development use.

Do not expose the HTTP server to untrusted networks. It has no authentication model yet and may expose indexed repository metadata through local APIs.

Please report security issues privately through GitHub Security Advisories. If advisories are unavailable, open a minimal issue saying that a private security contact is needed, without publishing exploit details.

Do not include secrets, private repository contents, tokens, or credentials in reports.
