# Contributing

SharpMemory is early and may change quickly.

Before contributing:

- keep changes small and focused;
- prefer existing project language: snapshots, segments, relationships, repositories, search;
- keep local data, databases, logs, and settings out of git;
- add tests for behavior that can regress.

By contributing to this project, you agree that your contributions are licensed under the Apache License 2.0.

Open an issue before large architectural changes.
